Crear Elementos con DOM - Interactivo

1. document.createElement(tag)

Crea un nuevo elemento HTML especificando la etiqueta.

2. document.createComment(text)

Crea un nodo de comentario (visible en el código fuente).

3. document.createTextNode(text)

Crea un nodo de texto.

4. node.cloneNode(deep)

Clona un nodo HTML (con hijos si deep es true).

5. node.isConnected

Verifica si un nodo está conectado al DOM.
