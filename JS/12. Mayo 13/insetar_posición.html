Insertar Elementos en el DOM

1. appendChild(node)

Añade un nodo como hijo del nodo actual.

2. insertAdjacentElement(pos, elem)

Inserta un elemento en una posición relativa al nodo.

Elemento de referencia

3. insertAdjacentHTML(pos, str)

Inserta HTML directamente en la posición indicada.

Elemento de referencia

4. insertAdjacentText(pos, text)

Inserta texto plano en una posición específica.

Elemento de referencia

5. insertBefore(newNode, referenceNode)

Inserta un nodo antes de otro como hijo del contenedor.

Nodo referencia
